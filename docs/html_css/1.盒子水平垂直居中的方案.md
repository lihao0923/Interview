# 盒子水平垂直居中的方案

## 定位：3种
### 1.absolute + margin // 需要知道宽高
<pre>
body {
    position: relative;
    height: 100%;
    overflow: hidden;
}

.box {
    width: 100px;
    height: 100px;
    background-color: green;

    position: absolute;
    top: 50%;
    left: 50%;
    margin: -50px 0 0 -50px;
}
</pre>


### 2.top, left, right, bottom + margin: auto // 需要有宽高，不用知道具体宽高是多少
<pre>
body {
    position: relative;
    height: 100%;
    overflow: hidden;
}

.box {
    width: 100px;
    height: 100px;
    background-color: green;

    position: absolute;
    top: 0;
    left: 0;
    bottom: 0;
    right: 0;
    margin: auto;
}
</pre>

### 3.absolute + transform // 不需要宽高，有兼容性问题
<pre>
body {
    position: relative;
    height: 100%;
    overflow: hidden;
}

.box {
    background-color: green;

    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}
</pre>


## display:flex // 有兼容性问题，移动端常用此方式
<pre>
body {
    display: flex;
    justify-content: center;
    align-items: center;

    height: 100%;
    overflow: hidden;
}
</pre>

## javascript
<pre>
    let HTML = document.documentElement,
        winW = HTML.clientWidth,
        winH = HTML.clientHeight,
        box = document.getElementById('box'),
        boxW = box.offsetWidth,
        boxH = box.offsetHeight;

    // 需要设置符合子body的position为relative
    box.style.position = 'absolute';
    box.style.left = (winW - boxW) / 2 + 'px';
    box.style.top = (winH - boxH) / 2 + 'px';
</pre>

## display: table-cell
<pre>
body {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
    // => 宽高不能是百分比,必须是固定宽高
}

.box {
    display: inline-block;
}
</pre>
